# Serverless curl-impersonate (serverless + docker + express + proxy + curl-impersonate)

Serverless HTTP proxy server configuration for curl-impersonate

## Getting started

```sh
pnpm yc:registry:create
pnpm yc:registry:configure

pnpm yc:ci
```
